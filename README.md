# 🎮 Tic Tac Toe

A simple and fun Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. Play locally with a friend and enjoy smooth, responsive gameplay in your browser.

## 🕹️ Features
- Two-player mode (Player X vs Player O)
- Win detection
- Game reset/restart button
- Clean UI with smooth user experience

## 🚀 Getting Started
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/tic-tac-toe.git
   ```
2. Navigate to the project folder:
   ```bash
   cd tic-tac-toe
   ```
3. Open `index.html` in your browser.

> No installation or dependencies required.
